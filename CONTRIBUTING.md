# Contributing to Istanbul Digital Nomads

Thanks for wanting to contribute! This community runs on the effort of its members.

## Ways to Contribute

**Share Knowledge** - Found a great coworking space or know the best way to get a residence permit? Add it to our [resources repo](https://github.com/istanbul-digital-nomads/resources).

**Organize Events** - Want to host a meetup or workshop? Check the [events repo](https://github.com/istanbul-digital-nomads/events) for templates. Just open an issue to propose an event.

**Improve Our Docs** - See a typo or outdated info? Pull requests are welcome on any repo.

**Help Newcomers** - Jump into the Telegram group and help people who just arrived.

**Suggest Ideas** - Open an issue in this repo. We're always looking to improve.

## How to Submit Changes

1. Fork the repo you want to contribute to
2. Create a new branch with a descriptive name
3. Make your changes
4. Submit a pull request with a clear description

## Content Guidelines

- Write in clear, simple English
- Include dates when info might become outdated
- Be specific: include addresses, links, and prices
- Avoid promotional content unless clearly labeled
- Use your own experience over generic advice

## Questions?

Open an issue or ask in the [Telegram group](https://t.me/istanbul_digital_nomads).
